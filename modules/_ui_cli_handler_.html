CLIDescriptor

CLIDescriptor: { handler: (...parameters: string[]) => string | Error; helpMessage: string; name: string; pattern: RegExp }
  • Defined in src/ui/cli-handler.ts:16

Description of a command line command, including a name, a help message, a RegExp pattern that matches the format of the command and extracts the parameters, and a handler.

The pattern should use RegExp capturing groups to capture command parameters. The handler, if invoked, will accept the string parameters (captured by pattern capturing groups, starting from index 1, i.e. without the full match), handles the command, and returns a string (if success) or an Error.

commandLineHandler

  • commandLineHandler(commands: CLIDescriptor[], enableHelp?: boolean): CommandLineHandler
    • Defined in src/ui/cli-handler.ts:29

    Returns a CommandLineHandler that handles commands according to an array of CLIDescriptor. Once the pattern of a CLIDescriptor matches the command line, the corresponding handler of the CLIDescriptor will be invoked to handle the command, and the result returned. If cannot find any CLIDescriptor in the array that matches the command, an error will be returned.

    Optionally (default true), the handler handles a 'help' command that reports the usage of the commands from the array of CLIDescriptor.

    Parameters

    • commands: CLIDescriptor[]
    • Default value enableHelp: boolean = true
